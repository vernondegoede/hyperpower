# Mollie Hyper plugin

Some extra motivation while developing apps using your Hyper terminal.

## Install using [HPM](https://github.com/zeit/hpm)

```
$ npm install -g hpm-cli
$ hpm install mollie-terminal
```

Restart Hyper.

## Install manually
1. Open `~/.hyper.js`.
2. Add `verminal` the the `plugins` array. 
3. Enjoy Verminal.
